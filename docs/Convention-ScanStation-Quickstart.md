# BallotTrack — Scan Station Quickstart (Convention Day)

**Tape this to the scanner.**

You scan ballots. You don't decide what counts — that's the Admin and the Judge. If anything is weird, **tell the Admin** and let them sort it out.

---

## 1. Set Up Your Station

1. Open the BallotTrack site → **Station Setup**.
2. Check the **agent banner** at the top:
   - 🟢 **Green** = scanner agent running. Good.
   - 🟡 **Yellow** = checking. Wait 10 seconds.
   - 🔴 **Red** = agent not running. **Double-click the BallotTrack Station shortcut on the desktop.** Wait for green.
3. You'll see a list of **active rounds**. Find your race + round (matches the paper color of the ballots in your stack).
4. Click **Start** on that round.

> If your round isn't in the list: the Admin hasn't opened tallying yet. Wait. Don't start without an assignment.

---

## 2. Wait for the Admin to Start a Pass

- The Scanner page shows: **"No active pass — waiting for admin to start a pass…"**
- When the Admin clicks **Start Pass N**, the page updates and you can begin.

---

## 3. Scan Ballots

1. Feed ballots through the scanner one at a time (front + back).
2. The agent picks up each ballot automatically — you'll see a confirmation per scan.
3. Watch your **Local** count climb. The Admin sees the **Total** across all stations.
4. Keep going until your stack is empty.

> You will **not** see vote tallies. That's intentional — you're not supposed to know who's winning while you scan.

---

## 4. When the Stack Is Done

- Tell the Admin you're finished.
- The Admin clicks **Complete Pass**. You're done with this pass.
- For Pass 2, repeat from Step 2.

---

## 5. Problem Ballots — Use the Spoiled Ballot Page

Click the **Spoiled Ballot** link on the Scanner page.

| Situation | Spoil Type | Notes |
|---|---|---|
| Ballot jammed or torn — won't feed | Unreadable / Jammed | Type the SN if you can read it. Snap a photo with the camera if helpful. |
| Voter clearly tried to undo their vote (scribbled out, "VOID" written, etc.) | Intent Undermined | Type the SN. Add notes describing what you saw. |

Click **Mark as Spoiled**. The ballot is removed from the count.

---

## 6. Things You Should NOT Do

- ❌ Don't start or complete a pass yourself (Admin only).
- ❌ Don't delete a pass. Ever. (Tell the Admin.)
- ❌ Don't try to "fix" a ballot that scanned wrong — flag it to the Admin so they can correct it on the Confirmation screen.
- ❌ Don't hand-type a ballot serial number into the scanner — if the QR won't read, the ballot lands in the **Flagged Review** queue and the Admin handles it.
- ❌ Don't scan ballots from a different round/color into your station — they'll get flagged "Wrong Round" and create cleanup work.

---

## Quick Troubleshooting

| Symptom | Do this |
|---|---|
| Agent banner is red | Double-click the desktop shortcut. Still red? Tell the Admin. |
| Round not in the list | Admin hasn't opened tallying yet. Wait. |
| Scanning but counts not going up | Check agent banner. If green, tell the Admin — could be a watch-folder problem. |
| Ballot jammed | Clear the jam. Use Spoiled Ballot → Unreadable/Jammed. |
| QR code damaged on the ballot | Just feed it — the agent flags it for review. Don't try to enter it manually. |
| Scanned a ballot from the wrong round by mistake | Tell the Admin immediately. Don't try to undo it yourself. |

**When in doubt: stop scanning and ask the Admin.**
